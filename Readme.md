# Placement Portal Application (PPA)

A full-stack web application built to manage campus recruitment processes efficiently.

## Problem Statement

Institutes require an efficient system to manage placement activities involving:

- Company approvals
- Placement drives
- Student registrations
- Application tracking

Manual processes using spreadsheets and emails create inefficiencies and tracking issues.  
This system provides a centralized digital solution.

---

## Tech Stack (As Per Mandatory Requirements)

### Backend
- Flask (REST API)
- SQLite (Database)
- Redis (Caching)
- Celery (Batch Jobs / Background Tasks)

### Frontend
- Vue.js
- Bootstrap (for styling)
- Jinja2 (only for entry point template)
